# devops-with-python

# scaffold
- makefile
- requirements.txt
- source code
- tests
- docker file

# Steps

1. Create python virtual environment `python3 -m venv .venv ` or `virtualenv .venv`
2. Activate .venv `source .venv/bin/activate`
3. Create empty files : `Makefile` `Dockerfile` `main.py` `src/__init__.py`
4. Populate `Makefile`
